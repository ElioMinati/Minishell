## Minishell ##

-- EPITECH PROJECT --

\\\\ Duration: 9 weeks total //

Reproduce linux terminal behavior.

Features:
  - Execute any linux command (emacs, ls, etc...)
  - Execute any local binary
  - Execute builtin commands (cd, setenv, unsetenv, env, exit)
  - Handle segfaults
  - Pipe handling (|)
  - Multiple pipe handling
  - Input redirection
